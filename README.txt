分数百分数练习工具 - 使用说明

1. 程序功能：
   - 第一轮：将1-20的数字的倒数转换为百分数
   - 第二轮：将百分数转换回分数形式
   - 两轮练习的顺序可选
2. 输入格式：
   - 百分数：可以输入 5.0% 或 5.00% 或 5%（自动补全%）
   - 分数：输入格式为 分子/分母，如 1/5

3. 运行要求：
   - Windows 7/8/10/11


4. 方法：
   双击运行百化分练习工具.exe


5. 注意事项：
   - 允许微小的计算误差
   - 按Ctrl+C可以中断程序
